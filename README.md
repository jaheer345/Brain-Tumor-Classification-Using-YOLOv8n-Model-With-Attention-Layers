# Brain-Tumor-Classification-Using-YOLOv8n-Model-With-Attention-Layers
This project uses the YOLOv8n model with SimAM Attention , Residual Attention and Tiled Squeeze-and-Excite(TSE) to classify glioma, meningioma, and pituitary tumors from MRI scans, providing a lightweight solution for real-time brain tumor detection.
